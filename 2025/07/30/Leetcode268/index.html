題目資訊

  • 題號：LeetCode 268
  • 難度：#Easy

題目說明

給定一個包含 n 個不同數字的陣列 nums，這些數字來自 0 到 n，找出陣列中缺少的那一個數字。

範例

🔍 解題思路

  1. 使用 range(n+1) 生成從 0 到 n 的所有數字。
  2. 對每個數字檢查是否存在於 nums 中，若不存在即為缺失的數字。
    ✅ 完整程式碼（原始解法）

📌 小結與優化

優點：

  • 記憶體使用率低（超過 80% 的解答）。

缺點：

  • 執行時間較長（僅超過約 10% 的解答），因為 if i not in nums 是 O(n) 操作。

優化建議：

  • 將 nums 轉換為 set，可以將查找時間從 O(n) 降為 O(1)。
  • 或使用數學總和公式、異或技巧達到 O(n) 時間與 O(1) 空間。
    🚀 優化範例：使用 set 提升查找效率
